Css Guide

  • * common.css에서 import하여 사용
    • - default.css : 초기화 및 기본 css 설정
    • - style.css : 디자인 스타일 설정
  • * 파일 위치 : assets/css/
/* font */ @import url("../fonts/NotoSans/notosans.css"); @import url("../fonts/Roboto/roboto.css"); /* style */ @import url("default.css"); /* 초기화 및 기본 css 설정 */ @import url("style.css"); /* 디자인 스타일 설정 */

  • * 기본 폰트 : Noto Sans, Roboto 사용
  • * 기본 사이즈 : 16px
